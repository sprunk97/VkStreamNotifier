# VkStreamNotifier
Отправляет рассылку о начале стрима на twitch.tv через приложение сообщества ВКонтакте

Файл credentials.json имеет вид:
```
{
 "twitch_id":"your_id",
 "twitch_token":"your_token",
 "twitch_username":"your_channel",
 "vk_app_id":"your_app_id",
 "vk_app_token":"your_app_token",
 "message":"sample_text",
 "list_ids":"list_of_ids_to_send",
 "vk_api_token":"another_app_token"
}
```
